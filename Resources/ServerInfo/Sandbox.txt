
[color=#ffffff]Параметры панели спавна объектов[/color]
[color=#a4885c]Default (По умолчанию)[/color] создает маленькие объекты, такие как кружки, без выравнивания, но выравнивает блочные объекты, такие как стены, по сетке.
[color=#a4885c]PlaceFree (Свободное размещение)[/color] создает все объекты без выравнивания.
[color=#a4885c]PlaceNearby (Размещение поблизости)[/color] ограничивает радиус появления примерно 2 плитками.
[color=#a4885c]SnapgridCenter (Привязка к центру сетки)[/color] выравнивает объект по центру плитки.
[color=#a4885c]SnapgridBorder (Привязка к границе сетки)[/color] выравнивает объект по границе плитки.

[color=#ffffff]Параметры выравнивания по сетке[/color]
[color=#a4885c]AlignSimilar (Выравнивание по подобным)[/color] привязывает объект к похожим объектам. В настоящее время не работает.
[color=#a4885c]AlignTileAny (Выравнивание по любой плитке)[/color] выравнивает объект по сетке.
[color=#a4885c]AlignTileEmpty (Выравнивание по пустой плитке)[/color] целевая плитка должна быть пустой.
[color=#a4885c]AlignTileNonDense (Выравнивание по неплотной плитке)[/color] в целевой плитке не должно быть коллайдеров.
[color=#a4885c]AlignTileDense (Выравнивание по плотной плитке)[/color] в целевой плитке должны быть коллайдеры.
[color=#a4885c]AlignWall (Выравнивание по стене)[/color] привязывается к вертикальным половинам плиток.
[color=#a4885c]AlignWallProper (Правильное выравнивание по стене)[/color] привязывает объект к середине краев плитки.
